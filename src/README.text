# Project Hibernate 2

## Опис

Проект Hibernate_2 включає в себе структуру бази даних для організації відеопрокату. У цьому проекті ми виявили деякі "ляпи" або недоліки у структурі бази даних, які можуть призвести до проблем при роботі з даними.

## Виявлені "ляпи"

1. Відсутність зовнішнього ключа у таблиці `film_text` на полі `film_id` таблиці `film`. Це може призвести до неконсистентності даних та складнощів при їхньому використанні.

## Пропозиції щодо виправлення

1. Додати зовнішній ключ у таблицю `film_text` на поле `film_id`, який посилається на первинний ключ таблиці `film`.

## Завершення

Цей файл README створений для опису виявлених недоліків у структурі бази даних проекту Video Store та пропозицій щодо їхнього виправлення. Будь-які подальші зміни у структурі бази даних повинні бути ретельно обгрунтовані та впроваджені з урахуванням найкращих практик проектування баз даних.
